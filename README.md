# Java Number Guessing Game

## Description
This is a Java console-based number guessing game where the computer generates
a random number and the user tries to guess it within limited attempts.

## Key Features
- Random number generation
- User input using Scanner class
- Provides hints (too high / too low)
- Loop-based game logic
- Beginner-friendly implementation

## Tech Stack
- Java
- Scanner class
- Random class
- Loops and conditional statements

## How to Run
1. Clone or download the repository
2. Compile the program:
   javac GuessNumberGame.java
3. Run the program:
   java GuessNumberGame

## Purpose
- Implement random number generation using Java Random class
- Enhance problem-solving and logical reasoning skills
- Create a fun beginner-level project suitable for portfolios and internships
- This project is created to practice Java basics, loops, and conditional logic for internship tasks.
